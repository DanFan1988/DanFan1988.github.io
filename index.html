My Projects

Hello Fellow (Backbone.js / Ruby On Rails): A Trello clone with boards, lists, cards, checklists, labels and smooth drag and drop functionality. Hello Fellow is a one-page application mainly built on Backbone with the database handled by Rails. It features very deeply nested views, sortable cards and lists that persist after refreshing, and an overridden backbone’s parse method to handled nested JSON data - which substantially reduced both the size and quantity of database queries. Live Github

Snake (JavaScript): A recreation of the classic game of snake made entirely from JavaScript. The game itself is completely agnostic of the UI that displays the game. A few new gameplay options were added in for user enjoyment. Live Github

Super ActiveRecord (Ruby): A clone of ActiveRecord’s Object Relational Mapping features. Extensive use of metaprogramming and SQL querries allowed for Mass Assignment, querying, persistence, and association methods Github
