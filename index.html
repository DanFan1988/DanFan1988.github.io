My Projects

Hello Fellow (Backbone.js / Ruby On Rails) | Live | Github

A Trello clone with boards, lists, cards, checklists, labels and smooth drag and drop functionality. Hello Fellow is a one-page application mainly built on Backbone with the database handled by Rails. It features very deeply nested views, sortable cards and lists that persist after refreshing, and an overridden Backbone parse method to handle nested JSON data - which substantially reduces both the size and quantity of database queries.

Snake (JavaScript) | Live | Github

A recreation of the classic game of snake made entirely from JavaScript. The game itself is completely agnostic of the UI that displays the game. A few new gameplay options were added in for user enjoyment.

Super ActiveRecord (Ruby) | Github

A clone of ActiveRecord’s Object Relational Mapping features. Extensive use of metaprogramming and SQL querries allowed for mass assignment, querying, association methods, and the full range of database interaction

ReRails (Ruby) | Github

Rewrote some basic functionality for Rail’s MVC framework, including a WEBrick server with cookies, Controller functionality that renders templates, a param’s hash that uses regex to handle nesting, and a router.
